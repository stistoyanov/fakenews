# Fake News - Музикално видео сценарий

Сценарий и таймлайн за музикалното видео на песента **"Fake News"** на групата **Обратен Ефект**.

## Описание

Това е проект за създаване на музикално видео за метъл/рок песента "Fake News", която сатирично коментира фалшивите новини и абсурдността на съвременния свят.

## Файлове

- **`Lyrics.md`** - Текстът на песента
- **`Lyrics-inspection.md`** - Детайлна разбивка на текста с визуални концепции за всеки ред
- **`Scene-Prompts.md`** - Детайлни промптове за всяка сцена (Main Prompt, Environment, Character Details, Camera Direction, Lighting, Visual Style, Mood & Emotion, Details & Effects)
- **`Character-Storylines.md`** - Сюжетни линии за всеки от четиримата членове на групата (Кольо, Мони, Йошо, Мартин) - какво се случва с всеки през цялата песен
- **`Storyboard.md`** - Storyboard с детайлни описания и готови AI промптове за генериране на скици за ключовите сцени
- **`Timeline.md`** - Таймлайн с таймкодове за синхронизация на визуалните елементи с музиката
- **`FakeNews.wav`** / **`FakeNews.mp3`** - Аудио файлове на песента
- **`analyze_audio.py`** - Python скрипт за анализ на аудио файла и генериране на таймкодове (приблизителни)
- **`sync_lyrics.py`** - Скрипт за автоматична синхронизация на текстове с аудио (използва вокална детекция)
- **`manual_sync.py`** - Интерактивен инструмент за **ръчна синхронизация** - най-прецизният метод
- **`create_video_simple.py`** - Python скрипт за създаване на примерен видеоклип с синхронизирани текстове

## Структура на песента

- **Встъп** (00:00 - 00:03)
- **Верс 1** (00:03 - 01:22) - 16 реда
- **Припев 1** (01:22 - 01:36)
- **Интерлюд** (01:36 - 01:47) - Фалшива новина
- **Верс 2** (01:47 - 02:27) - 8 реда
- **Припев 2** (02:27 - 02:41)
- **Припев 3** (02:41 - 02:55)
- **Финал** (02:55 - 03:08)

**Обща продължителност:** 03:08.019  
**Темпо:** 105.47 BPM

## Използване

За регенериране на таймкодове след промени в текста или аудио файла:

```bash
# Активиране на виртуалната среда
source venv/bin/activate

# Стартиране на анализа
python analyze_audio.py
```

### Създаване на примерен видеоклип

За създаване на примерен видеоклип с синхронизирани текстове:

```bash
# Активиране на виртуалната среда
source venv/bin/activate

# Създаване на примерен видеоклип
python create_video_simple.py
```

Това ще създаде файл `FakeNews_Sample.mp4` с:
- Синхронизирани текстове за всеки ред от песента
- Различни цветове за различните секции (Верс, Припев, и т.н.)
- Аудио от оригиналния файл

**Забележка:** Създаването на видеото може да отнеме няколко минути в зависимост от производителността на компютъра.

### Ръчна синхронизация (Препоръчително)

За най-прецизна синхронизация, използвай интерактивния инструмент за ръчна синхронизация:

```bash
# Активиране на виртуалната среда
source venv/bin/activate

# Стартиране на интерактивния инструмент
python manual_sync.py
```

**Как работи:**
1. Скриптът ще пусне песента
2. Слушай песента и натискай **SPACE**, когато започва всеки ред
3. Натисни **'r'** за повторение на текущия ред
4. Натисни **'s'** за запазване на прогреса
5. Натисни **'p'** за преглед на прогреса
6. Натисни **'q'** за изход

Алтернативно, ако нямаш pygame инсталиран, скриптът ще използва текстов режим, където въвеждаш таймкодовете ръчно (формат: MM:SS.mmm).

**Важно:** Ръчната синхронизация е най-прецизният метод и дава най-добри резултати!
